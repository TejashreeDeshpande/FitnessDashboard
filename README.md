# Fitness Dashboard

A modern, visually appealing Fitness Dashboard Android application built using **Jetpack Compose**. This project showcases custom UI components, advanced Canvas drawings, and a clean architecture approach.

## ✨ Features

- **Activity Ring:** A custom-drawn progress indicator using Compose Canvas API with a unique layered arc design.
- **Daily Stats:** At-a-glance view of Calories, Distance, and Active Time.
- **Health Metrics:** Track Heart Rate, Sleep duration, and Workout counts.
- **Goal Tracking:** Visual progress bars for daily steps, workout, and water intake goals.
- **Modern UI:** Dark-themed design with vibrant gradients and Material 3 components.

## 🛠 Tech Stack

- **Language:** [Kotlin](https://kotlinlang.org/)
- **UI Framework:** [Jetpack Compose](https://developer.android.com/jetpack/compose)
- **Architecture:** MVVM (Model-View-ViewModel)
- **Dependency Injection:** Manual DI (Clean and simple for this UI showcase)
- **Theme:** Material 3 with custom color palettes and typography.
